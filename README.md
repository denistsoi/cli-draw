# drawing program

https://github.com/denistsoi/draw

### prerequisites

- node latest (9.4.0)  

### installation

    $ cd draw
    $ npm install
    $ npm start

### running tests

    $ npm test

### assumptions

- Canvas can fit in memory (finite canvas)
- params (width/height || x/y coords) are non-negative integers
- rectangle feature could draw if within range of canvas (i.e. out of range?)
